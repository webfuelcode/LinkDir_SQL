Introduction

LinkDir is a simple web directory script to make an online site directory. This script is very simple and can be used for various purposes.

With your small team you have the ability to manage members to post upto 5 entries. Edit their entries or just delete.

Build a niche website on any topic like, earning list, crypto-currency site list, legit sites, online tools web list, photography or basic product list site or any other micro niche. Basically it is a directory script where you can manage members, add/edit category, posts.

Server Requirements

You will need to make sure your server meets the following requirements. Most of the web hosting providers give the ability to change or you can ask them.

  • PHP >= 7.2.5
  • BCMath PHP Extension
  • Ctype PHP Extension
  • Fileinfo PHP extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

Installation

Follow the below steps to get strated.

  • Unzip the source package.
  • Move files from the Public folder in the root. (Means under the public_html folder)
  • Put the other files in separate folder. (See the video for help)
  • When the uploading is done you have to make sure the index.php file point to the correct folder/directory.
  • For database settings, you have to open .env file. Which is a hidden file. Go to FTP settings to see hidden files.
  • And then enter the required details according to your host.
Enter your database details in DB_DATABASE, DB_USERNAME & DB_PASSWORD
One more setup is to fill the app detail. When a welcome mail is sent your site link will be present on that email. This is optional but required for branding.
Fill the APP_NAME="your site name" & APP_URL="your site URL"

Email settings

Another important part is to set the email server becasue LinkDir send welcome email to newly registered members on registration. Email is required when they request new password on forgot password page. The script will work fine if you do not set it. Find here free SMTP servers
Email sent for two purposes. First, when new members get registered and second, when they forget password and use the password reset page.
Gmail does provide free SMTP. If you have no two factor authentication, mailing will work instantly. Recommendation is to use the host's provided STMP server for better and easy setup.
